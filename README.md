# PDF Voice

iOS-приложение для чтения PDF вслух. Нативно, офлайн, бесплатно. Приоритет — русский язык.

План разработки и архитектура: [PLAN.md](PLAN.md).

## Статус

Основная функциональность готова, идёт подготовка к публикации в App Store.

- импорт PDF из Files, библиотека с сохранением позиции и закладками;
- просмотр через PDFKit с синхронной подсветкой звучащего предложения;
- текстовый слой → предложения (`NLTokenizer`) + **OCR сканов** через Vision (ru/en);
- чистка текста: колонтитулы, номера страниц, переносы, аббревиатуры, числа → слова;
- навигация: «Читать отсюда», скраббер страниц, сетка миниатюр;
- фон и экран блокировки (`AVAudioSession`, Now Playing, пульт/наушники);
- **два движка озвучки:** системный `AVSpeechSynthesizer` (офлайн) и нейросетевой
  **Silero** (русские голоса с ударениями) через свой сервер `tts.pdf-voice.com`;
  при недоступности сервера — беззвучный откат на системный голос.

Развёртывание Silero-сервера: [`silero-server/deploy/DEPLOY.md`](silero-server/deploy/DEPLOY.md).

## Сборка

> Нужен **полный Xcode** (App Store), не только Command Line Tools.

Проект описан через [XcodeGen](https://github.com/yonyz/XcodeGen) (`project.yml`) — `.xcodeproj` генерируется, а не хранится в репозитории.

```bash
brew install xcodegen      # один раз
cd pdf-voice
xcodegen generate          # создаёт PDFVoice.xcodeproj
open PDFVoice.xcodeproj
```

Дальше в Xcode: выбрать симулятор iPhone (или устройство) и ⌘R.

> Для запуска на реальном устройстве укажите свою команду подписи (Signing & Capabilities) — `DEVELOPMENT_TEAM` намеренно не зашит в `project.yml`.

## Структура

```
PDFVoice/
├─ App/        — точка входа
├─ Models/     — модель библиотеки (Codable)
├─ Library/    — хранилище (JSON) и экран списка
├─ Reader/     — экран чтения: PDFView + плеер
└─ Core/       — извлечение текста, движок речи, абстракция TTS
```
