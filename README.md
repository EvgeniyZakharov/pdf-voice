# PDF Voice

iOS-приложение для чтения PDF вслух. Нативно, офлайн, бесплатно. Приоритет — русский язык.

План разработки и архитектура: [PLAN.md](PLAN.md).

## Статус

**M1 — каркас + чтение текстового PDF** (текущая веха):
- импорт PDF из Files, библиотека с сохранением;
- открытие и просмотр через PDFKit;
- извлечение текстового слоя и разбиение на предложения (`NLTokenizer`, кириллица);
- озвучка нативным `AVSpeechSynthesizer` (русский голос Milena);
- плеер: play/pause, переход по предложениям, регулировка скорости;
- авто-прокрутка к текущей странице, сохранение/восстановление позиции;
- детект сканов без текстового слоя (полноценный OCR — M3).

## Сборка

> Нужен **полный Xcode** (App Store), не только Command Line Tools.

Проект описан через [XcodeGen](https://github.com/yonyz/XcodeGen) (`project.yml`) — `.xcodeproj` генерируется, а не хранится в репозитории.

```bash
brew install xcodegen      # один раз
cd pdf-voice
xcodegen generate          # создаёт PDFVoice.xcodeproj
open PDFVoice.xcodeproj
```

Дальше в Xcode: выбрать симулятор iPhone (или устройство) и ⌘R.

> Для запуска на реальном устройстве укажите свою команду подписи (Signing & Capabilities) — `DEVELOPMENT_TEAM` намеренно не зашит в `project.yml`.

## Структура

```
PDFVoice/
├─ App/        — точка входа
├─ Models/     — модель библиотеки (Codable)
├─ Library/    — хранилище (JSON) и экран списка
├─ Reader/     — экран чтения: PDFView + плеер
└─ Core/       — извлечение текста, движок речи, абстракция TTS
```
